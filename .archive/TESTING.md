### 测试指南

#### 在真机上测试

1. **准备工作**
   ```bash
   # 连接手机（开启 USB 调试）
   adb devices
   
   # 安装 APK
   cd src/android
   ./gradlew installDebug
   ```

2. **测试步骤**
   
   **测试 1: 连接中继服务器**
   - 打开 App
   - 进入"实例列表"
   - 点击"翎云"实例的"连接"按钮
   - 预期：状态变为 🟢，显示 clientId
   
   **测试 2: 发送消息**
   - 点击"翎云"实例卡片
   - 自动切换到"对话"界面
   - 输入"你好"
   - 点击发送
   - 预期：消息显示在列表中，收到回复
   
   **测试 3: 添加实例**
   - 进入"实例列表"
   - 点击右上角 "+" 按钮
   - 输入名称"测试实例"，地址"ws://localhost:8080"
   - 点击"确定"
   - 预期：新实例出现在列表中
   
   **测试 4: Sub-agent 监控**
   - 在对话界面
   - 发送一个会触发 sub-agent 的消息
   - 预期：底部显示 Sub-agent 面板
   - 点击展开
   - 预期：显示 Sub-agent 详情

3. **查看日志**
   ```bash
   adb logcat | grep -E "WebSocketManager|RelayClient"
   ```
